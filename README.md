# ts-mh
# 介绍
该项目为运行在[触动精灵](https://www.touchsprite.com/)下的`lua`脚本,对应的游戏是`梦幻西游手游`
# 功能
1. 满足切换账号,选择服务器,目前仅支持第一个角色
2. 可以运行`刮刮乐`,`师门`,`宝图`,`秘境`,`运镖`,`带队捉鬼`,`混队捉鬼`,`科举`,`三界奇缘`的功能

# 主要变更
1. 支持游戏闪退后的恢复
2. 改用多线程
